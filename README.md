# Bank Fraud Analysis: Exploratory Data Analysis (EDA)

## Project Overview
This project investigates a synthetic financial dataset to identify statistical markers of fraudulent behavior. By utilizing Python and advanced data visualization, I isolated key variables that correlate with suspicious transactions to improve detection accuracy.

## Technical Skills Used
*   **Python**: Primary language for data cleaning and analysis.
*   **Pandas & NumPy**: Used for data manipulation and structured analysis.
*   **Seaborn & Matplotlib**: Used to generate statistical visualizations and fraud heatmaps.
*   **GitHub**: Version control and project deployment.

## Key Insights
*   Identified specific transaction types (e.g., 'TRANSFER' and 'CASH_OUT') where fraud was most prevalent.
*   Discovered that fraudulent transactions often involved the entire balance of the origin account.
